### [Portfolio Sync](https://www.npmjs.com/package/portfolio-sync)

- **Auth** — Form, HTTP Basic, Bearer Token, Cookies, Custom Script
- **Testing** — node:test (natif), MD5 hash diff
- **UX** — Chalk, Ora spinners, dotenv

```bash

```

## Ce portfolio

### Stack

- **Framework** — Vue 3 (Composition API, `<script setup>`)
- **Build** — Vite 6
- **Routing** — Vue Router 4
- **i18n** — vue-i18n (FR/EN)
- **Styling** — Tailwind CSS 4
- **Animations** — GSAP (ScrollTrigger, ScrollToPlugin, TextPlugin)
- **3D** — Three.js
- **SEO** — @unhead/vue + pre-rendering script custom
- **Hosting** — Vercel

### Scripts

```bash
npm run dev              # Dev server
npm run build            # Build + injection meta
npm run build:prerender  # Build + pré-rendu HTML pour SEO
npm run preview          # Preview du build
npm run update-screenshots  # Capture via portfolio-sync
```

### Structure

```
src/
├── App.vue
├── main.js
├── assets/

├── router/

config/
├── i18n.js
└── locales/        # fr.json, en.json

scripts/
├── inject-meta.mjs
└── prerender.mjs
```
